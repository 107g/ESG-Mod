Types of prerequisites:

CHECK SCHEMA FOR IF THESE ARE VALID

MetaPrerequisite - Example: <MetaPrerequisite Operator="Or">, Operator can be Or or And, MetaPrerequisites have multiple prerequisites inside of them
DatePrerequisite - Example: <DatePrerequisite>01/21 00:00:00 AM;01/25 11:59:59 PM</DatePrerequisite>
GameSettingPrerequisite - Example: <GameSettingPrerequisite    Flags="Prerequisite,Discard">AllowSuperWeapons,True</GameSettingPrerequisite>
SupportPrerequisite - Example: <SupportPrerequisite Flags="Cancel" Politics="Politics01">1</SupportPrerequisite>, no idea what the "1" is for
TechnologyPrerequisite - Example: <TechnologyPrerequisite Flags="Technology,Discard">TechnologyUnknown</TechnologyPrerequisite>
InterpreterPrerequisite - Example: <InterpreterPrerequisite Flags="Prerequisite,Discard">Property(Context,@../ClassEmpire, SystemCount) gt 1</InterpreterPrerequisite>, see interpreter.md for usage
PathPrerequisite - Example: <PathPrerequisite Flags="Prerequisite">AcademyFactionDifficultyEasy</PathPrerequisite>, Checks for a simulation descriptor at a path
PathValidityPrerequisite - No examples, no idea what this does
DownloadableContentPrerequisite - Example: <DownloadableContentPrerequisite Flags="Discard,Edition">DLCHisshos</DownloadableContentPrerequisite>, Checks if a DLC is installed
DiplomaticRelationStatePrerequisite - Example: <DiplomaticRelationStatePrerequisite Flags="Discard" Inverted="true">DiplomaticRelationStateUnkown</DiplomaticRelationStatePrerequisite>
FleetPrerequisites - Example:
<FleetPrerequisites>
  <InterpreterPrerequisite Inverted="false" Flags="RequireMilitaryPower">SumProperty(Context, @'ClassGarrisonFleet/ClassShip',OffensiveMilitaryPower) gt 0</InterpreterPrerequisite>
</FleetPrerequisites>, Checks for fleet values
TutorialLevelPrerequisite - Example: <TutorialLevelPrerequisite>Advanced</TutorialLevelPrerequisite>
QuestStatePrerequisite - Example:  <QuestStatePrerequisite Flags="Prerequisite" Inverted="true" QuestDefinitionName="AcademyQuest03-B-Coop" QuestState="InProgress"/>
Prerequisites - Example:         
<Prerequisites Target="$(Empires)" AnyTarget="true">
  <QuestStatePrerequisite Flags="Prerequisite" Inverted="true" QuestDefinitionName="Deeds_Regular_EmpireDevelopment_Stage4_B" QuestState="InProgress"/>
  <QuestStatePrerequisite Flags="Prerequisite" Inverted="true" QuestDefinitionName="Deeds_Regular_EmpireDevelopment_Stage4_B" QuestState="Completed"/>

  <QuestStatePrerequisite Flags="Prerequisite" QuestDefinitionName="Deeds_Regular_EmpireDevelopment_Stage4_B" QuestState="NotStarted"/>
  <QuestStatePrerequisite Flags="Prerequisite" QuestDefinitionName="Deeds_Regular_EmpireDevelopment_Stage4_C" QuestState="NotStarted"/>
            
  <QuestStatePrerequisite Flags="Prerequisite" Inverted="true" QuestDefinitionName="Deeds_Regular_EmpireDevelopment_Stage4_C" QuestState="InProgress"/>
  <QuestStatePrerequisite Flags="Prerequisite" Inverted="true" QuestDefinitionName="Deeds_Regular_EmpireDevelopment_Stage4_C" QuestState="Completed"/>
</Prerequisites>, pretty much just metaprerequisites but with a forced "and"
